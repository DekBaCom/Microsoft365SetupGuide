# Microsoft 365 Setup Guide

A comprehensive, step-by-step guide for deploying and configuring Microsoft 365 — covering Tenant Configuration, Azure AD, Microsoft Endpoint Manager (Intune), and Collaboration Governance.

Built with [Jekyll](https://jekyllrb.com/) and the [Just the Docs](https://just-the-docs.com/) theme, hosted on GitHub Pages.

**Live site:** [https://DekBaCom.github.io/Microsoft365SetupGuide](https://DekBaCom.github.io/Microsoft365SetupGuide)

---

## Contents

- **Tenant Configuration** — Custom domains, organization settings, EOP & ATP baseline, alert policies
- **Azure AD Configuration** — Combined MFA/SSPR registration, Conditional Access baseline policies
- **Microsoft Endpoint Manager** — Device enrollment, compliance policies, App Protection (MAM), Autopilot
- **Collaboration Governance** — DLP, retention policies, sensitivity labels, SharePoint & Teams governance

---

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Then open [http://localhost:4000/Microsoft365SetupGuide](http://localhost:4000/Microsoft365SetupGuide)
